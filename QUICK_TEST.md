## 🐛 如果遇到问题

### 问题 1: 快捷键不生效

**解决方案**：
1. 打开 **系统设置** → **隐私与安全性** → **辅助功能**
2. 点击 `+` 添加 Xcode 或 SimpleClipboard
3. 勾选启用
4. 重新运行应用

---

### 问题 2: 编译错误

**检查**：
1. Xcode 版本是否 >= 15.0
2. macOS 版本是否 >= 13.0
3. 查看错误信息（红色文本）

**常见错误**：
- `Cannot find type 'ClipboardManager'` → Clean Build Folder (`⌘ + Shift + K`)
- `Module not found` → 重新打开项目

---

### 问题 3: 密码仍被记录

**调试**：
1. 在 Xcode 控制台查看日志
2. 检查 `ClipboardManager.swift:50` 的 `isSensitive()` 方法
3. 添加调试日志：
   ```swift
   print("Testing password:", isSensitive("Test@123456"))
   ```

---

### 问题 4: 历史记录不保存

**调试**：
1. 查看控制台是否有 "💾 Saved" 日志
2. 检查 UserDefaults：
   ```swift
   print(UserDefaults.standard.array(forKey: "clipboardHistory") ?? [])
   ```

---
